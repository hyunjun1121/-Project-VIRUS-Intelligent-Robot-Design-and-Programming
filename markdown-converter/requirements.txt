# Markdown Converter Tool 의존성 패키지
# pandoc은 별도로 시스템에 설치해야 합니다.

# 기본 Python 라이브러리들만 사용하므로 별도 requirements는 없습니다.
# 단, pandoc이 시스템에 설치되어 있어야 합니다.

# pandoc 설치 방법:
# Windows: https://pandoc.org/installing.html#windows
# macOS: brew install pandoc
# Ubuntu/Debian: sudo apt-get install pandoc
# CentOS/RHEL: sudo yum install pandoc

# PDF 변환을 위한 LaTeX 엔진도 필요합니다:
# Windows: MiKTeX 또는 TeX Live
# macOS: MacTeX
# Ubuntu/Debian: sudo apt-get install texlive-xetex
